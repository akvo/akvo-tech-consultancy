# Flow Email Notification Jobs 

Blast email from flow data transformed into html template.

## Usage

```bash
$ export $KEYCLOAK_USER="YOUR_KEYCLOAK_USER"
$ export $KEYCLOAK_PWD="YOUR_KEYCLOAK_PWD"

$ pip install -r requirements.txt
$ python app.py
```
